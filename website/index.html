Setup

One Compose stack. Four commands.

Docker Compose v2 is the only requirement. GPU acceleration is optional.

$ git clone https://github.com/MacNite/Mathom-House---Your-Local-AI-Memory-House.git
$ cd Mathom-House---Your-Local-AI-Memory-House
$ cp .env.example .env
$ make up
$ make models
✓ Mathom is ready at http://localhost:8080
